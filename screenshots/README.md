## How to Generate Screenshots

1. Run the development server: `npm run dev`
2. Open http://localhost:3000
3. Use browser dev tools to capture screenshots at various breakpoints:
   - Desktop: 1920x1080
   - Tablet: 768x1024
   - Mobile: 375x812

4. Save to this directory with descriptive names

## Asset Guidelines

- Use PNG format for static images (better quality)
- Use GIF or WebM for animations
- Optimize images before committing (use TinyPNG or similar)
- Maximum file size: 500KB per image
- Use descriptive filenames
